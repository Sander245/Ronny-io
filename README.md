# RONNY IO

Multiplayer tank battle game built with Node.js and Socket.IO.

## Features
- 62 unique tank types with upgrade paths
- Real-time multiplayer combat
- Polygon enemies for XP grinding
- Advanced stat system
- Cheat/debug menu for testing

## Local Development

1. Install dependencies:
```bash
npm install
```

2. Start server:
```bash
node server.js
```

3. Open browser:
```
http://localhost:13126
```

## Deployment

See [DEPLOYMENT.md](DEPLOYMENT.md) for instructions on hosting with GitHub Pages + free server hosting.

## Controls
- **WASD** - Move
- **Mouse** - Aim
- **Left Click** - Shoot
- **E** - Auto-fire toggle
- **Mouse Wheel** - Zoom
- **N** - Toggle health bars

## Cheat Menu
Type `devcheats` to activate:
- **G** - Give XP (adjustable with +/-)
- **B** - Max stats
- **I** - Toggle invincibility
- **T** - Teleport all players to cursor
- **R** - Reset tank
